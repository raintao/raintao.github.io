# 基于vue-cli搭建的pc端vue框架，所用技术为vue+vue-router+vuex+axios+less。 集成了bootstrap，elemet-ui,jquery,md5,jquery.cookie.js等工具

# 目录介绍：


				static文件夹主要放一些图片,第三方文件。src中的目录如下：  


                |-- components  单文件组件文件夹
                |-- diy         封装的api 接口等
                |-- main.js     入口文件
                |-- view       页面
                |-- public      css重置、统一制定主题、样式等css文件,字体图标
                |-- router      路由文件
                |-- store       vuex
                |-- app.vue

# 模版搭建

 $ git clone https://github.com/raintao/vue2_pc_template.git

 $ cnpm install

 $ npm run dev

 $ npm run build
