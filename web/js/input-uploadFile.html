# input用H5的formData上传多个文件

# XMLHttpRequest Level 2添加了一个新的接口FormData.利用FormData对象,我们可以通过JavaScript用一些键值对来模拟一系列表单控件,我们还可以使用XMLHttpRequest的send()方法来异步的提交这个"表单".比起普通的ajax,使用FormData的最大优点就是我们可以异步上传多个二进制文件.
    html代码如下：

    <label class="label" for="uploadFujian">
      上传附件
      <form style="display:none;"  class="file" enctype="multipart/form-data">
        <input type="file" multiple="multiple" accept="*"  name="vuefile" id="uploadFujian">
      </form>
    </label>
    <progress class="progress"></progress>
    js代码：

      $("#uploadFujian").change(function(){
        var that=this,formData=new FormData();
        var files= $(that).get(0).files
        if(files.length>0){
          for(var i=0;i<files.length;i++){
            formData.append("vuefile"+i,files[i]);
          }
        }
        $.ajax({
          type:'POST',
          url:"上传的地址",
          xhr:function(){
            var myXhr= $.ajaxSettings.xhr();
            if(myXhr.upload){
              myXhr.upload.addEventListener('progress',_this.progressHandle,false);
            }
            return myXhr;
          },
          data:formData,
          cache:false,
          contentType:false,
          processData:false,
          success:function(data){
            //你的代码
          },
          error:function(err){
            //错误处理
          }
        });
        progressHandle(e){
          if(e.lengthComputable){
            $(".progress").attr({value:e.loaded,max:e.total});
          }
        }
},
# 其中progressHandle函数是配合显示上传进度用的
